paragraf adalah kumpulan dari beberapa kalimat yang saling mendukug.

selain belajar paragraf di materi ini juga mengenalkan tag hr, gunanya untuk membuar garis lurus (horizontal), lanjut selain tag P ada juga tag pre contohnya sebagai berikut: 

        ini adalah tag pre digunakan untuk menampilkan teks atau paragraf 
        dalam format yang sudah kita tentukan di file HTML 
                seperti di microsoft world ketika menggunakan tab akan ada indentasi
                kalau di HTML menggunakan tag pre
